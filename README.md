# Movie Ratings Data Analysis 🎬

A Python-based data analysis project that explores movie datasets to extract trends, top-rated films, and genre popularity. This project utilizes the MovieLens dataset to perform data cleaning, merging, and visualization.

## 📊 Project Overview
This project analyzes movie ratings to answer key questions:
* What are the highest-rated movies with a significant number of reviews?
* How are ratings distributed across the dataset?
* Which movie genres are the most popular among users?

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas (Data Manipulation), Matplotlib & Seaborn (Visualization)
* **Environment:** GitHub Codespaces / Local VS Code

## 📂 Dataset Structure
The project uses two primary files:
1. `movies.csv`: Contains `movieId`, `title`, and `genres`.
2. `ratings.csv.gz`: A compressed version of the ratings dataset containing `userId`, `movieId`, `rating`, and `timestamp`.

> **Note:** The ratings file is compressed to bypass GitHub's 25MB file size limit.
